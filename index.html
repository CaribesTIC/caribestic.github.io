Vitest (VI)

Es un marco de prueba de unidad ultrarrápido. Permite la creación de un corredor simple que no necesita lidiar con la complejidad de transformar archivos de origen y puede enfocarse únicamente en proporcionar el mejor DX durante las pruebas.

Vue Test Utils (VTU)

¡La biblioteca oficial de utilidades de prueba para Vue.js! Es un conjunto de funciones de utilidad destinadas a simplificar las pruebas de los componentes de Vue.js. Proporciona algunos métodos para montar e interactuar con los componentes de Vue de forma aislada.

Vue Testing Library (VTL)

Vue Testing Library se basa en DOM Testing Library al agregar API para trabajar con componentes de Vue. Está construido sobre VTU, la biblioteca de prueba oficial para Vue, ajustando algunos métodos de ambas fuentes.

Vue Testing Handbook (VTH)

Vue Testing Handbook es un sitio que contiene informaciòn relevante para aprender sobre Vue(TDD).
